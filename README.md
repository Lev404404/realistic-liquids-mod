# Realistic Liquids Mod

[![Minecraft](https://img.shields.io/badge/Minecraft-1.21.1-brightgreen)](https://minecraft.net)
[![Fabric](https://img.shields.io/badge/Loader-Fabric-blue)](https://fabricmc.net)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

Реалистичный мод для Minecraft 1.21.1 Fabric, добавляющий физику воды и лавы с эффектами волн, частиц и кастомных шейдеров.

## 🌊 Возможности

- ✅ **Реалистичная вода** с волнами и брызгами
- ✅ **Реалистичная лава** с эффектами огня и дыма
- ✅ **Кастомные GLSL шейдеры** для обеих жидкостей
- ✅ **Система частиц** (капли воды, искры лавы)
- ✅ **Волновая физика** с распространением
- ✅ **Специальные рецепты крафта** на верстаке
- ✅ **Возврат в ведро** при клике на жидкость
- ✅ **Полная конфигурация** прозрачности и эффектов
- ✅ **JSON конфиг** для настроек

## 📋 Рецепты Крафта

### Реалистичная Вода
```
L L L
L W L
L L L
```
**W** = Ведро воды | **L** = Лазурит

### Реалистичная Лава
```
R R R
R L R  
R R R
```
**L** = Ведро лавы | **R** = Редстоун

## 📥 Установка

### Требования
- Java 17+
- Fabric Loader 0.15.11+
- Minecraft 1.21.1
- Fabric API 0.97.2+

### Шаги установки

1. **Скачайте JAR файл** из [Releases](https://github.com/Lev404404/realistic-liquids-mod/releases)
2. **Откройте папку** `.minecraft/mods`
3. **Скопируйте** `realistic-liquids-mod-1.0.0.jar` туда
4. **Запустите** Minecraft с Fabric Loader
5. **Создавайте** реалистичные жидкости!

## ⚙️ Конфигурация

После первого запуска создастся файл: `config/realistic_liquids.json`

```json
{
  "water": {
    "transparency": 0.7,
    "waveHeight": 0.5,
    "waveSpeed": 1.0,
    "waterDropletSize": 2,
    "enableWaves": true,
    "enableParticles": true,
    "enableShader": true
  },
  "lava": {
    "transparency": 0.6,
    "emitHeight": 0.3,
    "emberParticleCount": 5,
    "enableFlames": true,
    "enableSmoke": true,
    "enableShader": true
  }
}
```

### Параметры воды:
- **transparency** (0.0-1.0) - Прозрачность воды
- **waveHeight** (0.0-1.0) - Высота волн
- **waveSpeed** (0.0-2.0) - Скорость волн
- **waterDropletSize** (1-10) - Размер капель
- **enableWaves** - Включить волны
- **enableParticles** - Включить частицы брызг
- **enableShader** - Включить шейдер

### Параметры лавы:
- **transparency** (0.0-1.0) - Прозрачность лавы
- **emitHeight** (0.0-1.0) - Высота выбросов
- **emberParticleCount** (1-20) - Количество искр
- **enableFlames** - Включить пламя
- **enableSmoke** - Включить дым
- **enableShader** - Включить шейдер

## 🎮 Использование

### Создание реалистичной воды:
1. Положите **ведро воды** в центр верстака
2. Окружите его **лазуритом** (8 блоков)
3. Получите **ведро реалистичной воды**
4. Вылейте как обычное ведро

### Создание реалистичной лавы:
1. Положите **ведро лавы** в центр верстака
2. Окружите его **редстоном** (8 блоков)
3. Получите **ведро реалистичной лавы**
4. Вылейте как обычное ведро

### Вычерпывание жидкости:
- **Кликните пустой рукой** по блоку жидкости
- Жидкость исчезнет с эффектом частиц
- **Ведро вернётся в инвентарь**

## 🎨 Особенности реализации

### Шейдеры (GLSL 1.20)
- **Вода:** Динамические волны, преломление света, кольябления
- **Лава:** Пульсирующее свечение, анимированный поток, горячий оранжевый цвет
- **Оба:** Используют функции `sin()` и `cos()` для плавной анимации

### Система частиц
- **Капли воды:** SPLASH частицы с физикой гравитации
- **Искры лавы:** LAVA частицы с восходящей траекторией
- **Затухание:** Плавное исчезновение по времени

### Волновая физика
- Система распространения волн между соседними блоками
- Демпфирование с коэффициентом 0.99
- Натяжение жидкости и распространение
- Автоматическая очистка затухших волн
